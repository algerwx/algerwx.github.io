Redisson 是什么？

  • 定位：基于 Redis 的 Java 分布式服务框架，提供高阶 API 简化开发
  • 核心价值
    • 将分布式锁、队列、集合等复杂功能封装为类似 JDK 的 API
    • 自动管理连接、序列化、故障恢复，减少样板代码

快速使用方式

1. 依赖引入（Maven）

<dependency>
  <groupId>org.redisson</groupId>
  <artifactId>redisson-spring-boot-starter</artifactId>
  <version>3.18.0</version>
</dependency>

2. 配置 (Spring boot)

application.yaml

spring:
  redis:  # Redisson 专有配置
    redisson:
      file: classpath:singleRedisson.yaml

singleRedisson.yaml

singleServerConfig:
  address: "redis://127.0.0.1:6379"
  connectionMinimumIdleSize: 5
  connectionPoolSize: 50
  connectTimeout: 3000
  timeout: 5000

3. 注入 redisson 对象

@Autowired
private RedissonClient  redissonClient;

4. 分布式锁案例

public void redisLockOne() {

    RLock lock = redissonClient.getLock("lock");
    int tryTimes = 0;
    try {
        while (!lock.tryLock()) {
            tryTimes++;
            Thread.sleep(2000);
            System.out.println("尝试获取锁: "+tryTimes);
        }
        System.out.println("获取锁成功");
        redisLockTwo();
        Thread.sleep(10000);
    }catch (Exception e){
        e.printStackTrace();
    }finally {
        lock.unlock();
        System.out.println("释放锁");
    }

}

使用 redisson 很方便的实现了分布式锁。

常见操作

​​字符串（String）- RBucket

  • 核心方法

    • set(value): 存储字符串或对象 自动序列化
    • get(): 获取值
    • expore(duration,timeUnit): 设置过期时间

  • 示例

public void set(@RequestBody RedisParm redisParm) {  
    String key = redisParm.getKey();  
    String value = redisParm.getValue();  
    RBucket<String> bucket = redissonClient.getBucket(key);  
    bucket.set(value, 30, TimeUnit.SECONDS); // 存储并设置30秒过期  
}

Hash - RMap

  • 核心方法

    • put(key,value): 添加字段
    • get(key): 获取值
    • expore(duration,timeUnit): 设置整个哈希的过期时间

  • 示例

public void setHash() {  
    RMap<String, Integer> userMap = redissonClient.getMap("user:2");  
    userMap.put("age", 30);  
    int age = userMap.get("age");  
    System.out.println(age);  
}

列表（List） - RList

  • 核心方法
    • add(element)：添加元素
    • get(index)：获取元素
    • remove(element)：删除元素
  • 示例
RList<String> taskList = redissonClient.getList("tasks"); taskList.add("Task1"); 
String firstTask = taskList.get(0);

高级分布式功能​

分布式锁（Lock） - RLock

  • ​​核心方法​​：
    • tryLock(waitTime, leaseTime, timeUnit)：非阻塞尝试加锁
    • lock()：阻塞式加锁
    • unlock()：释放锁
  • ​场景​​：秒杀库存扣减、定时任务排他执行。
  • ​​示例​：
public void redisLockOne() {  
  
    RLock lock = redissonClient.getLock("lock");  
    int tryTimes = 0;  
    try {  
        while (!lock.tryLock()) {  
            tryTimes++;  
            Thread.sleep(2000);  
            System.out.println("尝试获取锁: "+tryTimes);  
        }  
        System.out.println("获取锁成功");  
        redisLockTwo();  //可重入
        Thread.sleep(10000);  
    }catch (Exception e){  
        e.printStackTrace();  
    }finally {  
        lock.unlock();  
        System.out.println("释放锁");  
    }  
  
}  
  
private void redisLockTwo() {  
    RLock lock = redissonClient.getLock("lock");  
    System.out.println("redisLockTwo 尝试获取到锁");  
    int tryTimes = 0;  
    try {  
        while (!lock.tryLock()) {  
            System.out.println("redisLockTwo 尝试获取到锁" + tryTimes++);  
        }  
        System.out.println("redisLockTwo 获取到锁");  
    }catch (Exception e){  
        e.printStackTrace();  
    }finally {  
        lock.unlock();  
        System.out.println("释放锁");  
    }  
  
}

布隆过滤器 - RBloomFilter

  • ​方法​：

    • tryInit(expectedInsertions, falseProbability)：初始化
    • add(element), contains(element)

  • ​场景​：大规模数据去重（如爬虫URL判重）。

  • 示例

RBloomFilter<String> filter = redisson.getBloomFilter("url_filter"); 
filter.tryInit(1000000L, 0.03); // 预期100万数据，3%误判率 
filter.add("https://example.com"); 
boolean exists = filter.contains("https://example.com");

延迟队列 - RDelayedQueue

  • 方法​​：offer(element, delay, timeUnit)

  • ​场景​​：订单超时关闭、延迟消息通知。

  • 示例

RQueue<String> queue = redissonClient.getQueue("delayed_tasks");  
RDelayedQueue<String> delayedQueue = redissonClient.getDelayedQueue(queue);  
delayedQueue.offer("check_order", 40, TimeUnit.SECONDS);
